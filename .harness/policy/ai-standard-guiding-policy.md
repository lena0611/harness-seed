# AI Standard Guiding Policy Reference

이 문서는 `ai-standard/docs`에 있는 최상위 길라잡이를 가리키는 참조 문서입니다.

원본:
- GitLab: `https://git.smartscore.kr/ai-standard/docs`
- 경로: `standards/ai-standard-guiding-policy.md`

원칙:
- 최상위 정책의 원본은 `ai-standard/docs`에서 관리합니다.
- `harness-seed`는 설치기이므로 원본 정책을 소유하지 않습니다.
- 하네스, 스택 기준, 템플릿, 에이전트 도구를 바꿀 때는 원본 정책의 위배 여부를 먼저 확인합니다.

핵심 관점:
- 프롬프트는 의도를 전달하지만, 하네스는 실수했을 때 피해를 제한합니다.
- AI 에이전트는 빠른 실행력을 가진 불완전한 자동화 도구로 보고, 주변에 테스트, 제한, 관찰, 복구, 차단 장치를 둡니다.
- 프로젝트 기준은 실제 버그 수정과 기능 개발 중 발견한 반복 패턴을 근거로 성장합니다. 하네스는 도메인 규칙을 임의로 발명하지 않고, 기존 코드와 사용자 확인을 바탕으로 로컬 룰 후보를 승격합니다.
- 실제 업무 진행 보고에서 에이전트 사고 흐름은 원시 내부 추론이 아니라 요청, 컨텍스트, 영향, 실행, 판단, 검증의 visible trace로 요약합니다. 단순 질문 응답, 잡담, 메타 확인처럼 업무 진행 보고가 아닌 턴에는 trace 형식을 강요하지 않습니다.
- 프로젝트 기준이 길어지면 항상 모두 읽지 않고 인덱스, 최신 요약, 에이전트 판단 컨텍스트로 좁힙니다.

충돌 리포팅 요약:
- 충돌 출처는 회사 공통, 스택, 템플릿, 프로젝트, 개인 기준 중 어디인지 명시합니다.
- 충돌 대상은 코드, 문서, manifest, lock, CI, 런타임 설정 중 무엇인지 명시합니다.
- 충돌 의미는 안전 차단, 검증 누락, 구조 불일치, 스타일 차이, 런타임 차이, 문서-코드 불일치 중 무엇인지 설명합니다.
- 선택지는 프로젝트 기준 승격, 템플릿/스택 기준 수용, 예외 문서화, 개인 기준 분리, 판단 보류 중 하나로 정리합니다.
- 선택 결과는 `decision-log.md`, `developer-input-queue.md`, `waivers.json`, 프로젝트 기준 문서 중 알맞은 곳에 남깁니다.

충돌 해석 순서:
1. 회사 공통 필수 차단 기준
2. 사용자의 명시 지시
3. 프로젝트 기준
4. 템플릿 사용 계약
5. 스택 기준
6. 회사 공통 기본 운영 기준
7. 개인 기준
8. 에이전트 기본값

회사 공통 필수 차단 기준은 보안, 권한, 검증, 복구, 기록, 차단처럼 피해를 제한하는 기준입니다. 회사 공통 기본 운영 기준은 프로젝트/템플릿/스택 기준으로 구체화될 수 있는 운영 기본값입니다.

작업 전 확인:
1. 회사 공통 기준에 특정 스택 내용을 섞고 있지 않은가?
2. 스택 기준이나 템플릿이 회사 공통 기준의 프로세스를 우회하지 않는가?
3. 프로젝트 기준이나 기존 전용 하네스를 덮어쓰지 않는가?
4. 개인 기준을 팀 공유 기준보다 우선시키고 있지 않은가?
5. 에이전트 작업에서 검증 흐름을 선택 사항으로 낮추고 있지 않은가?
6. 회사 공통 필수 차단 기준과 회사 공통 기본 운영 기준을 구분하고 있는가?
7. 사용자의 명시 지시나 프로젝트 기준이 필수 차단 기준을 낮추는 방향으로 해석되고 있지 않은가?
8. 기준이 쌓이는 순서와 충돌 해석 순서를 혼동하고 있지 않은가?
9. 템플릿 가이드를 프로젝트 로컬룰로 복사하거나 템플릿 계약이 프로젝트 기준을 대체하게 만들고 있지 않은가?
10. 충돌이 생겼을 때 출처, 영향, 선택지, 기록 위치를 개발자가 알 수 있도록 리포트하고 있는가?
11. 임시 예외를 허용한다면 범위, 사유, 만료 조건을 남길 수 있는가?
12. 프로젝트 런타임 설정을 하네스 편의를 위해 덮어쓰고 있지 않은가?
13. 생성 컨텍스트를 원본 기준처럼 취급하거나 깨진 참조로 남기고 있지 않은가?
14. 업데이트 경로가 최신성뿐 아니라 버전 추적과 복구 가능성도 제공하는가?
15. 메이저 업데이트나 기준 해석이 바뀌는 변경을 명시적 선택 없이 자동 적용하고 있지 않은가?
