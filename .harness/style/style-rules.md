# 스타일 규칙

저장소 전체에 공통으로 적용하는 기본 코딩 스타일입니다. 프레임워크 또는 SFC 같은 스택 의존 규칙은 활성 스택의 instructions에서 다룹니다.

## JavaScript / TypeScript 공통
- 문자열은 기본적으로 single quote를 사용합니다.
- 세미콜론은 사용하지 않습니다.
- import는 정렬 상태를 유지합니다.
- 불필요한 빈 줄을 누적하지 않습니다.
- 파일 끝에는 개행을 유지합니다.

## 스타일과 구조 규칙의 우선순위
- 스타일 규칙은 구조 규칙보다 우선하지 않습니다. 아키텍처 계층 규칙을 깨면서 스타일을 맞추지 않습니다.
- 활성 스택이 추가하는 스타일(예: SFC `script setup`)은 해당 스택 instructions를 따릅니다.

## 주석
- 필요한 경우에만 주석을 추가합니다.
- 주석은 한국어로 작성합니다.

## 주의
- 스타일 규칙은 현재 자동화 범위 내에서 검증되는 것부터 우선 적용합니다.
- 의미적 판단이 필요한 스타일은 lint보다 리뷰/정책 판단 영역으로 남을 수 있습니다.
